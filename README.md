# Protect the President – SA-MP Gamemode

A simple **Protect the President** gamemode for San Andreas Multiplayer.

## Gameplay

* One player becomes the President
* Secret Service must protect the President
* Assassins attempt to eliminate the President

## Features

* Random President selection
* Team assignment
* Weapon loadouts
* Basic win condition system

## Installation

1. Download the repository
2. Place `protect_president.pwn` in `/gamemodes/`
3. Compile with **PAWNO**
4. Add to `server.cfg`

```
gamemode0 protect_president
```

5. Start the SA-MP server.

## Requirements

* SA-MP Server
* Pawn compiler (PAWNO)
