# Protect the President — SA-MP Gamemode

A lightweight **Protect the President** minigame for **San Andreas Multiplayer (SA-MP)**.
Players are divided into teams where one player must survive while others protect or eliminate them.

## Gameplay

* One player is randomly selected as the **President**
* **Secret Service** players must protect the President
* **Assassins** attempt to eliminate the President
* The round ends when the President is killed or the timer expires

## Features

* Random President selection
* Automatic team assignment
* Basic weapon loadouts
* Round timer and win conditions
* Simple and easy to modify

## Installation

1. Place `protect_president.pwn` inside your server's `gamemodes/` folder
2. Compile the script using **PAWNO**
3. Add the gamemode to `server.cfg`

```
gamemode0 protect_president
```

4. Start the SA-MP server

## Requirements

* SA-MP Server
* PAWN Compiler (PAWNO)
